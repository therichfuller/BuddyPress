== Description ==

A collection of small tweaks for the activity stream.

* Force login to view comments (select which activity types)
* Disable replying on selected activity types (+ admins) - ie blocked activity_comment will disable nested replies
* Disable favoriting on selected activity types
* Add CSS classes to each activity entry depending on friendship, following, self
* Add View buttons to activity entires (ie, View Blog Post, View Activity Status)
